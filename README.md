# A Star Trek-themed memory game built to practise functional components in React

# [Live](https://therealseanwallace.github.io/weatherPlusPlus)

    
## Learning objectives:
    
  Develop skills in:

  * React functional components
  * State
  * Hooks
  * Lifecycle methods
